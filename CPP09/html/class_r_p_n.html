◆ evaluateExpression()

void RPN::evaluateExpression ( const std::string & expression)
static

Evaluates a mathematical expression in Reverse Polish Notation (RPN).

This method takes a string representing an RPN expression and evaluates it using a stack. It supports basic arithmetic operators: +, -, *, /. The method reads tokens from the expression, pushes numbers onto the stack, and applies operators to the top elements of the stack.

Parameters
expressionThe RPN expression to be evaluated.
Exceptions
StackErrorif the expression is invalid or contains invalid operators.
